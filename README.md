# 💰 Controle Financeiro

Sistema de controle financeiro desenvolvido em **Python** para gerenciamento de receitas, despesas, saldo e movimentações através do terminal.

---

## 📋 Funcionalidades

- ➕ Cadastro de receitas
- ➖ Cadastro de despesas
- 📊 Consulta de saldo
- 📝 Histórico de movimentações
- ❌ Exclusão de movimentações
- 📈 Resumo financeiro
- ✅ Validação de entradas

## ▶️ Como executar

Clone o repositório:

```bash
git clone https://github.com/cauarpmarques/Controle_Financeiro.git
```

Entre na pasta:

```bash
cd SISTEMA_DE_CONTROLE_FINANCEIRO_(CLI)
```

Execute o projeto:

```bash
python main.py
```
